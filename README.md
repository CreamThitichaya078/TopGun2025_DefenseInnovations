# TopGun2025 Drone Detection, Tracking, and Localization using AI

โปรเจคนี้พัฒนาระบบตรวจจับ ติดตาม และคำนวณตำแหน่งของโดรนด้วย AI

## รายละเอียด
ใช้ YOLOv8 ในการตรวจจับและติดตามโดรน จากนั้นนำข้อมูล bounding box ไปใช้เป็นฟีเจอร์สำหรับโมเดล Random Forest เพื่อทำนายตำแหน่งละติจูด ลองจิจูด และความสูงของโดรน พร้อมสาธิตการทำงานบน Raspberry Pi

## เทคโนโลยีที่ใช้
- YOLOv8  
- Deep Learning  
- Machine Learning  
- Random Forest  
- Raspberry Pi
